Dynamic Load Balancing Using SDN
In this project, we provide a dynamic load management strategy for SDNs to optimise link utilisation in DCNs while taking flow priority into account. The programme estimates the cost of each link and determines the shortest routes between each host and the other hosts. When a way becomes congested, the optimal alternative route with the lowest connection cost and the lowest traffic flow takes its place.

Virtual TV
The main goal of this project was to attempt to overlay a subject video onto another video that already included the appropriate frame for playing the second subject film. The user was required to define the four corners of the frame.
